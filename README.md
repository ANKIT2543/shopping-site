# E-Commerce Frontend Clone (Amazon) 🛒

A static frontend replication of the Amazon homepage. This project focuses entirely on UI/UX structure, demonstrating the ability to recreate complex, real-world web layouts from scratch using semantic HTML and CSS.

## 🚀 Technical Highlights
* **Complex CSS Layouts:** Extensively utilized CSS Flexbox and CSS Grid to recreate the navigation bar, hero banner, and multi-column product card sections.
* **Semantic Structure:** Built using proper HTML5 semantic tags to ensure the page is well-structured and accessible.
* **Pixel-Perfect Styling:** Focused on replicating exact spacing, typography, and hover effects to match the production-level application.

## 🛠️ Tech Stack
* **Languages:** HTML5, CSS3
* **Concepts:** Flexbox, CSS Grid, Box Model, Responsive Layouts
